# Features
* Starting/Pausing/Reseting the game
* Color selection
* Grid resizement
* Update interval selection
* Generation counter

# Usage
Prerequisites: Qt5
```
git clone https://github.com/bjorked/game-of-life/
cd game-of-life
qmake game-of-life-qt.pro
make
./game-of-life-qt
```
